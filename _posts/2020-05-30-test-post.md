##### &#128640; Async & Await

자바스크립트의 비동기 처리패턴중 가장 최근에 나온 문법.

기존 Promise의 문법의 단점을 보완하고 가독성좋은 코드로 표현가능.

-------------------------
문법:



```
async function 함수명(){
    await 비동기처리할함수명();
}

async function test() {
    await test1();
}
```
